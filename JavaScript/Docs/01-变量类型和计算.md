#### 变量类型  变量计算
基本数据类型存储在栈内存中；
引用数据类型存储在堆内存中，存储在栈内存中的是该数据类型在堆内存中的地址。
+ 值类型 
```javascript
var a=20;
var b=a;
b=21;
console.log(a); //20
```

+ 引用类型  对象、数组、函数
特点：无限制扩展属性。
```javascript
var a={age:20};
var b=a;
b.age=21;
console.log(a.age);//21
```

#### typeof 运算符得到的结果
基本数据类型：`string`,`number`,`boolean`,`null`,`undefined`
`typeof`只能区分值类型和函数类型，不能区分引用类型(除了函数类型)

+ string
+ boolean
+ number
+ undefined
+ object------({}和null，是object类型)
+ function
